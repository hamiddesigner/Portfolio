# Hamid Ali Portfolio

A clean, modern portfolio website designed to showcase design work and professional experience.

## Features

- ✨ Clean and minimal design
- 📱 Fully responsive layout
- 🎨 Pixel-perfect implementation from Figma design
- ⚡ Interactive FAQ section
- 🎯 Featured work showcase
- 📧 Contact section with calendar integration
- 🔗 Social media links

## Design

This portfolio has been built to match 100% with the [Figma design](https://www.figma.com/design/S2CBcILdwG8THilwlNS4Nr/Portfolio?node-id=151-510&t=uTEOh0IkV8Ct9d81-11).

### Design Specifications

- **Font**: Inter (400, 500, 600, 700)
- **Primary Colors**: 
  - Black: `#000000`
  - White: `#ffffff`
  - Gray shades: `#666666`, `#808080`, `#f5f5f5`, `#fafafa`
  - Accent colors for project cards: `#dedef2`, `#d9edeb`, `#fae3d9`

### Layout

- Desktop width: 640px content area
- Padding: 30px vertical, 400px horizontal (desktop)
- Gap between sections: 74px

## File Structure

```
Vs-code-portfolio/
├── index.html          # Main HTML structure
├── styles.css          # All styling (converted from Tailwind)
├── script.js          # Interactive functionality
└── README.md          # Project documentation
```

## Sections

1. **Navigation** - Clean header with navigation links and profile avatar
2. **Hero** - Introduction with call-to-action buttons
3. **Featured Work** - Showcase of 3 major projects
4. **Skills & Experience** - Tag cloud of skills and technologies
5. **FAQ** - Interactive accordion for common questions
6. **Contact** - Email and calendar booking integration
7. **Footer** - Social links and copyright

## Usage

Simply open `index.html` in a web browser. No build process or dependencies required.

```bash
open index.html
```

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Responsive Breakpoints

- Desktop: 1440px+
- Tablet: 768px - 1024px
- Mobile: 480px - 768px
- Small mobile: < 480px

## Interactive Features

- FAQ accordion toggle
- Smooth hover effects on buttons and cards
- Active navigation state
- Responsive navigation menu

## Customization

To customize the portfolio:

1. **Update content**: Edit text in `index.html`
2. **Change colors**: Modify CSS variables and color values in `styles.css`
3. **Add images**: Replace placeholder images with your project screenshots
4. **Update links**: Replace social media and contact links with your own
